# Парсер Красное & Белое

## Запуск

```bash
python main_kb.py
```

## Структура

Парсер обходит категории:
- Вино
- Шампанское/Игристое
- Коньяк
- Виски
- Водка
- Пиво
- Сопутствующие товары
- Крепкие напитки

## Режимы работы

### С LLM (Ollama)
```env
USE_LLM=true
LLM_PROVIDER=ollama
```

### Без LLM (традиционный)
```env
USE_LLM=false
```

## Выходные данные

Файлы в `output/`:
- `kb_{timestamp}.csv` - товары
- `parser.log` - лог
